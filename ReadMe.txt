Git is a version control system that logs everything.
Git 是一種「全部留下紀錄」的版本控制系統。

To easy the burden of management,
為了減輕我管理上的麻煩，

Please only edit files of your own.
請各位同學「只」編輯自己的檔案。

File naming format:
檔案命名格式為：

StudentID_HW01.py
學號_HW01.py

Change the "StudentID" into yours, 
其中「學號」請改為自己的學號，

and the number in HW01 according to the assignment.
HW01 中的數字，則依作業指定更動。

The assignment will be given as:
作業發佈時將採以下格式：

Assignment_HW01.py

Use "git pull" command often to keep yourself updated with the class.
常使用 "git pull" 指令保持和全班同學同步。 ^.^ 晚來風疾
